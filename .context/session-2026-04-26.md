# Expensif ‚Äî Session Summary (2026-04-26)

## Project
A Go + SQLite expense tracking web app with server-rendered HTML UI and JSON API.

## Stack
- **Language:** Go 1.25
- **Database:** SQLite via `modernc.org/sqlite` (pure Go, no CGO)
- **CSS:** Tailwind CSS via CDN with `darkMode: 'class'`
- **Templates:** Go `html/template` with shared components

## Files

### Go Source
| File | Purpose |
|------|---------|
| `main.go` | HTTP server setup, routing |
| `db.go` | SQLite init, migrations |
| `expense.go` | Data models, CRUD, preferences, currency symbols |
| `handlers.go` | JSON API handlers |
| `templates.go` | HTML page handlers, template parsing, `basePageData` |

### Templates (`templates/`)
| File | Purpose |
|------|---------|

## Schema

```sql
CREATE TABLE expenses (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    amount REAL NOT NULL,
    category TEXT NOT NULL,
    description TEXT NOT NULL,
    date TEXT,
    currency TEXT DEFAULT 'USD',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE preferences (
    id INTEGER PRIMARY KEY CHECK (id = 1),
    currency TEXT DEFAULT 'USD',
    dark_mode INTEGER DEFAULT 0
);
```

## Routes

### HTML
- `GET /` ‚Äî Expenses list
- `GET /daily` ‚Äî Daily timeline view
- `GET /preferences` ‚Äî Settings page
- `POST /preferences` ‚Äî Save settings
- `GET /expenses/new` ‚Äî Add form (supports `?date=` prefill)
- `POST /expenses/new` ‚Äî Create (actions: `save` ‚Üí `/`, `another` ‚Üí `/expenses/new`)
- `GET /expenses/edit/{id}` ‚Äî Edit form
- `POST /expenses/edit/{id}` ‚Äî Update
- `POST /expenses/delete/{id}` ‚Äî Delete

### JSON API
- `GET /api/expenses` / `POST /api/expenses`
- `GET /api/expenses/{id}` / `PUT /api/expenses/{id}` / `DELETE /api/expenses/{id}`
- `GET /api/categories` ‚Äî Distinct categories used in last 3 months, ordered by frequency
- `GET /api/summary` ‚Äî Total + by category

## Features Built

1. **CRUD** ‚Äî Create, list (100), get by ID, update, delete expenses
2. **Per-transaction currency** ‚Äî 17 currencies with symbols (MYR, USD, JPY, CNY, THB, EUR, GBP, SGD, KRW, AUD, CAD, INR, VND, PHP, IDR, HKD, TWD)
3. **Category autosuggest** ‚Äî Fetch `/api/categories`, render clickable pills below category input; sorted by most used in last 3 months
4. **Humanized dates** ‚Äî `Today`, `Yesterday`, then `humanize.Time()` fallback; raw date in `title` attribute
5. **Daily View** ‚Äî Cards per day, expenses table inside, daily total, "+ Add expense" link pre-filled with that date
6. **Preferences (server-side)** ‚Äî Currency + dark mode persisted in SQLite; loaded at startup; applied to all pages via `basePageData()`
7. **Dark mode** ‚Äî Full `dark:` Tailwind variants on all templates
8. **Shared form** ‚Äî Add/edit use same template via `dict` helper; add has extra "Create & add another" button
9. **Form field order** ‚Äî Amount ‚Üí Description ‚Üí Category ‚Üí Date
10. **Validation** ‚Äî Amount, description, category required on both HTML and JSON

## Key Design Decisions
- **No SPA** ‚Äî Server-rendered pages with standard form posts
- **No JS frameworks** ‚Äî Vanilla JS only (fetch for category pills)
- **SQLite in home dir** ‚Äî `~/.expensif/expenses.db`
- **Preferences as singleton row** ‚Äî `id = 1` constraint
- **Per-transaction currency** overrides preference currency for display; totals are naive sums (no conversion)
